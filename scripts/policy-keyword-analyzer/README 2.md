# Policy Documents Folder

Place your privacy policy files here for analysis.

## Supported Formats
- `.txt` - Plain text files
- `.md` - Markdown files  
- `.html` / `.htm` - HTML files

## Recommended Sources

Download privacy policies from a variety of companies:

### Big Tech
- Google, Facebook/Meta, Apple, Microsoft, Amazon, Twitter/X

### Streaming
- Netflix, Spotify, Disney+, YouTube, Twitch

### E-commerce
- eBay, Etsy, Shopify, PayPal, Stripe

### Social Media
- TikTok, Snapchat, LinkedIn, Reddit, Pinterest

### Startups & Niche
- Various B2B SaaS companies
- Gaming platforms
- Health/Fitness apps
- Financial services

## Tips

1. **Save as plain text** when possible - cleaner for analysis
2. **Include diverse sources** - different writing styles help
3. **Mix languages** (optional) - helps identify universal patterns
4. **Include non-policies** - helps test the detector's accuracy

## File Naming

Name files descriptively:
- `google_privacy_policy_2024.txt`
- `stripe_terms_of_service.txt`
- `unknown_startup_privacy.md`
